# Python介绍

- 诞生时间: 1989
- 发明人: Guido van Rossum (荷兰)
- 特点:
  - 解释性语言, 代码开源
  - 语法简单,代码优雅
  - 第三方库多
  - 相比C/C++ 和 Java , Python运行速度较慢

- 应用领域:
  - web开发-->Django
  - 游戏开发-->Pygame;结合C/C++做大型游戏
  - 科学计算-->Numpy,
  - 人工智能-->Tensorflow
  - 云计算,大数据 --> OpenStack...
